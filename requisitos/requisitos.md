# 📋 Requisitos da Gestão de TI

## ✅ Requisitos Funcionais

RF01 - O sistema deve permitir registro de incidentes(chamados) 
RF02 - O sistema deve permitir categorização de chamados  
RF03 - O sistema deve permitir definição de prioridade  
RF04 - O sistema deve permitir acompanhamento de status  
RF05 - O sistema deve permitir geração de relatórios gerenciais  
RF06 - O sistema deve permitir gestão de SLAs  
RF07 - O sistema deve permitir criação de catálogo de serviços  

---

## ⚙️ Requisitos Não Funcionais

RNF01 - O sistema deve possuir alta disponibilidade (mínimo 99%)  
RNF02 - O tempo de resposta deve ser inferior a 2 segundos  
RNF03 - O sistema deve garantir segurança da informação (controle de acesso)  
RNF04 - Interface amigável e intuitiva  
RNF05 - Compatibilidade com dispositivos móveis  
RNF06 - Estar na faixa de orçamento aceitável

---

## 🔐 Requisitos de Segurança

- Controle de acesso baseado em perfis (RBAC)
- Registro de logs de auditoria
- Criptografia de dados sensíveis
